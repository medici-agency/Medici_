Повноцінне рішення для управління згодою на cookies з підтримкою GDPR, CCPA, категорій згоди, блокування скриптів та аналітики.

== Опис ==

**Medici Cookie Notice** — це сучасний плагін для WordPress, який забезпечує повну відповідність вимогам GDPR, CCPA та іншим законам про конфіденційність.

= Основні функції =

* **Кастомізований банер** — позиціонування, кольори, анімації, тексти
* **Категорії cookies** — necessary, analytics, marketing, preferences
* **Автоматичне блокування скриптів** — Google Analytics, Facebook Pixel, YouTube, тощо
* **Google Consent Mode v2** — повна інтеграція з GA4, GTM, Google Ads
* **Гео-детекція** — автоматичне визначення юрисдикції (GDPR, CCPA, LGPD)
* **Журнал згод** — збереження історії для аудиту
* **Аналітика** — статистика прийняття/відмови по категоріях та регіонах
* **Twemoji іконки** — красиві SVG емоджі замість стандартних

= Преміум функції включені =

* Consent Logs (журнал згод) з експортом у CSV
* Analytics Dashboard зі статистикою
* Script Blocker з патернами для популярних сервісів
* Geo Detection для GDPR/CCPA
* Google Consent Mode v2
* A/B тестування банерів
* Мультидоменна підтримка

= Сумісність =

* WPML / Polylang
* WP Super Cache, W3 Total Cache, WP Rocket
* Elementor, Divi, Gutenberg
* Contact Form 7, WooCommerce

== Встановлення ==

1. Завантажте плагін в директорію `/wp-content/plugins/`
2. Активуйте плагін через меню 'Плагіни' в WordPress
3. Перейдіть до Налаштування > Cookie Notice для конфігурації

== Часті запитання ==

= Чи плагін відповідає GDPR? =

Так, плагін підтримує всі вимоги GDPR:
- Opt-in згода до встановлення cookies
- Гранулярний вибір категорій
- Можливість відкликання згоди
- Збереження доказів згоди
- Відображення політики конфіденційності

= Чи підтримується Google Consent Mode v2? =

Так, плагін повністю інтегрований з Google Consent Mode v2 для GA4, GTM та Google Ads.

= Чи блокуються скрипти до отримання згоди? =

Так, Script Blocker автоматично блокує скрипти за патернами (Google Analytics, Facebook, YouTube, тощо) до отримання відповідної згоди.
